NOMBRE DEL PROYECTO
Cuestionario Autoevaluativo — Inteligencia Artificial · Manual 2026

TIPO DE PROYECTO
Plantilla web interactiva de autoevaluación tipo test.

DESCRIPCIÓN GENERAL
Plantilla HTML autocontenida para crear cuestionarios formativos interactivos. La demo incluye 30 preguntas de opción múltiple basadas en un documento ficticio sobre cómo la inteligencia artificial puede cambiar la vida, el trabajo, la educación, la sanidad, la economía, la creatividad, la regulación y otros ámbitos sociales.

El recurso incluye temporizador, retroalimentación inmediata, barajado aleatorio de respuestas, navegación entre preguntas, barra de progreso, contador de aciertos y pantalla final con resultados personalizados.

PERSONALIZACIÓN DEL CUESTIONARIO
Para cambiar las preguntas:
1. Abrir el archivo .html con un editor de texto o código.
2. Localizar el array const QUESTIONS = [...].
3. Editar cada objeto de pregunta.
4. Modificar el enunciado en el campo q.
5. Modificar las opciones en el array o.
6. Modificar el índice de la respuesta correcta en el campo a.
7. Modificar la explicación de acierto en el campo ok.
8. Modificar la explicación de error en el campo ko.

Para cambiar el logo:
Sustituir el string base64 en el atributo src de la imagen con clase logo.

Para cambiar el tiempo:
Modificar la constante const DURATION = 10*60;
El valor está expresado en segundos. Por ejemplo, 15*60 equivale a 15 minutos.

Para cambiar el número de preguntas:
Añadir o eliminar objetos dentro del array QUESTIONS. El sistema adapta automáticamente el progreso, los resultados y la revisión final.
